Exemplo de Tipos de Display

display: block;

Este é um elemento com display block. Ele ocupa toda a largura disponível.

display: inline; Este é um elemento com display inline. Ele fica na mesma linha que o texto.
display: inline-block;
Este é um elemento com display inline-block. Ele tem largura definida, mas fica em linha.
display: flex;
Este é um elemento flexível dentro de um container flex.
Ele permite controlar o layout e alinhamento dos elementos filhos.
display: grid;
Este é um elemento em um layout de grade.
Ele organiza os elementos em linhas e colunas.
Este é um item adicional da grade.
E aqui temos mais um item da grade.
Você pode ajustar a posição e o tamanho dos elementos na grade.
